# Scorpio's Nix-On-Droid Configuration

This repo is an ever evolving and changing Nix-On-Droid Configuration. I
am still working on learning bot Nix and Home-Manager in an effort to 
eventually port my system to a more permanat NixOS install. For the time
being though, Nix-On-Droid serves as both an effective learning tool and
testing grounds for my experiments. Feel free to poke around and use what
you feel works, I'll vouch for nothing in terms on stability at this point.
